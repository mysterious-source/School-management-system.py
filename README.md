# School-management-system.py
This project allows users to store and analyze student information, including academic marks across multiple subjects.
The School Management System is designed to manage student academic performance data.

The system allows users to store, view, and analyze student information and subject marks.

This project was developed as a learning exercise to improve programming logic, data management, and software development skills.

The application provides the following functionalities:

Add student names and academic marks

Manage multiple subjects including:

Mathematics

Computer Science

Physics

Chemistry

Biology

History

Geography

ICT

Economics

Business

View student lists

Display student marks

Calculate student average performance

Generate simple academic reports
